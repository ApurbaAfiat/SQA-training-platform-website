# SQA Training Platform Website

An interactive single-page training platform for junior QA engineers.

This project is designed to help learners practice core Software Quality Assurance (SQA) skills through guided content and hands-on simulations, including API testing, Postman workflows, Newman command usage, bug reporting, and basic security testing concepts.

## What This Project Includes

- API basics training content (methods, status codes, request/response structure)
- Postman fundamentals with practical workflow guidance
- Live mock API tester with request simulation and auto-validation
- Test script playground for Postman-style test logic practice
- Newman CLI simulation panel and command references
- GitHub issue reporting training with a built-in issue board
- Intro-level API security testing (VAPT) concepts
- Daily task tracker with XP/progress feedback
- Cheatsheets for fast revision

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript (no external framework)

## Run Locally

Because this is a static website, no build process is required.

1. Clone the repository:

```bash
git clone https://github.com/ApurbaAfiat/SQA-training-platform-website.git
```

2. Open the project folder:

```bash
cd SQA-training-platform-website
```

3. Open the main file in your browser:

- Open index.html directly, or
- Use VS Code Live Server extension for a local server preview

## Learning Goals

This project helps learners build practical confidence in:

- Understanding API behavior and response validation
- Writing and organizing test cases in Postman
- Using Newman for CLI-based collection execution
- Reporting clear, reproducible bugs in a professional format
- Performing basic API security checks during QA testing

## Notes

- This is a training and simulation platform, not a production backend system.
- Mock endpoints are implemented in front-end JavaScript for learning purposes.
